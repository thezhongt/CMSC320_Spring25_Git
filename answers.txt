Q1: Why might we consider using Git?
Answer: Git is a great way to monitor and ensure proper version control.  It provides an easy way to keep track of many different versions each with different changes.

Q2: What field could data science be applied to? Give an Example.
Answer: Data science can be applied to pretty much every field! Being able to analyze data and make predictions is always useful.  For example, a great usage of data science within the financial sector could be predicting stock prices based on historic data.
