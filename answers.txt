Q1: Why might we consider using Git?

Answer: Git is a great way to monitor and maintain version control.  With Git, we can ensure that there are no issues combining versions if we have multiple people working on them, etc.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science can be applied to pretty much any field because anything can benefit from analyzing data to make predictions.  An example in the financial sector is predicting stock prices.
